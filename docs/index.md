# NNSR - Neural Signal Reader

欢迎使用NERCN Neural Signal Reader (NNSR) 文档!

## 简介

NNSR是一个专为神经信号处理和分析设计的开源工具包。它提供了从数据采集到信号处理、特征提取和分析的全套解决方案。

## 主要功能

- **在线分析**：实时处理和可视化神经信号
- **离线分析**：深入分析已记录的数据集
- **多格式支持**：兼容多种神经信号数据格式
- **可扩展性**：模块化设计，易于扩展和定制
